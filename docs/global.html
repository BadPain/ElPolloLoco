clearAllTrackedTimers()

Clears all tracked intervals and timeouts. This function iterates over all stored interval and timeout IDs, clears them using clearInterval and clearTimeout respectively, and then resets the global storage arrays to an empty state. It ensures that no pending intervals or timeouts remain active.
